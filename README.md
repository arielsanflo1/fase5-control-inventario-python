## Descripcion

Este repositorio contiene una solucion en Python que recorre una matriz de inventario y calcula la cantidad exacta que debe pedirse de cada articulo cuando el stock actual es menor que el stock minimo requerido.

## Estructuras utilizadas

- Matriz implementada como lista de listas.
- Ciclo `for` para recorrer el inventario.
- Condicionales `if/else` para clasificar el estado.
- Funciones para calcular el pedido y generar el informe.

## Ejecucion

```bash
python problema_3_inventario.py
```

No requiere librerias externas.

## Resultado esperado

El programa presenta el detalle de seis articulos y un resumen con:

- 6 articulos auditados.
- 4 articulos por reabastecer.
- 25 unidades totales por solicitar.
